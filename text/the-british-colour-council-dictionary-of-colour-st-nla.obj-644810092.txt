Code word for British Colour Council
BECEC
Code word for this card
BYCYC
The purpose of this Dictionary of Colour
Standards is to simplify work in connection
with colour throughout all colour-using
industries so that the standard name or
standard number will always signify the
colour so designated in this Dictionary.
Comparison with the Standards should be
A
made in a good north light.
When quoting from this card, which shows
colours Nos. B.C.C. I—B.C.C.1 — B.C.C . 240, the
name, number and code word should be
given.

is limited to the cost of this Dictionary
should any errors or omissions in its
preparation, printing or distribution be
made, and the Dictionary is accepted
subject to the condition.
